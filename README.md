## Pokedex

### How to run the gateway

switch to the folder & install dependencies:

```
cd backend
npm i
```

to run the localhost:

```
npm run dev

```

it runs on http://localhost:4000

### How to run the frontend

switch to the folder & install dependencies:

```
cd frontend
npm i
```

to run the localhost:

```
npm run dev
```

it runs on http://localhost:3000
